🎯 How It Works

This system monitors Nifty 50 index in real-time and sends Telegram alerts when bullish signals are detected based on EMA (Exponential Moving Average) analysis.

🔍 Signal Detection:

🟢 Bullish Signal: When 5-minute candle is fully above EMA(5)

🔗 API Endpoints

GET /.netlify/functions/status - System status and health check
GET /.netlify/functions/auth - One-time Upstox authentication setup
GET /.netlify/functions/check_alerts - Manual alert check (also triggered by GitHub Actions)

⚙️ Setup Instructions

  1. Click "Setup Authentication" above
  2. Login to your Upstox account
  3. Copy the access token and add to environment variables
  4. System will automatically start monitoring
  5. Receive lifetime alerts in your Telegram!
